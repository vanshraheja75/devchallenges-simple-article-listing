## Overview

![Thumbnail for the Simple Article Listing coding challenge](./thumbnail.jpg)

This is my responsive solution for the Simple Article Listing challenge. The goal was to create a clean, organized layout for displaying blog posts, including a featured article section, matching the design specifications.

### Built With

-   HTML5 (Semantic Markup)
-   CSS3 (Flexbox & CSS Grid)
-   Responsive Design

## Features

-   **Responsive Layout:** Uses CSS Grid/Flexbox to adjust the article arrangement for mobile, tablet, and desktop views.
-   **Semantic HTML:** properly structured content using `<article>`, `<header>`, and `<main>` tags.
-   **Visual Accuracy:** closely mimics the spacing, typography, and card shadows of the provided design.
